JavaScript & Skriptsprachen – Einordnung

Web-Programmierung · Vorlesung 1 · Grundlagen

In der Vorlesung wurde JavaScript als Skriptsprache im Browser eingeordnet. Wichtig war dabei der Vergleich zu klassischen Programmiersprachen und zu Auszeichnungssprachen.

Sprachtypen (Kurzüberblick)

  • Auszeichnungssprachen: beschreiben Struktur/Inhalt (z. B. HTML)
  • Programmiersprachen: eher „strenger“, oft kompiliert (z. B. Java, C++)
  • Skriptsprachen: flexibel, häufig interpretiert (z. B. JavaScript)

Was ist typisch für Skriptsprachen?

  • werden meist interpretiert (kein klassischer Compile-Schritt wie bei Java)
  • oft dynamisch typisiert (Typen ergeben sich zur Laufzeit)
  • günstig für schnelle Iteration/Prototyping

Rolle von JavaScript im Web

  • läuft direkt im Browser (Client) – z. B. über <script>
  • macht Seiten „lebendig“: DOM-Manipulation, Events, Interaktion
  • Grundlage für moderne Webapps (und später auch für Frameworks)
Merke:
HTML beschreibt was angezeigt wird, CSS wie es aussieht, und JavaScript was passiert (Logik/Interaktion).
