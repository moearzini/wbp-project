wbp-project

Öffentliches GitHub-Repository

Dieses Repository enthält den vollständigen Quellcode zum Web Programming Project (WBP). Es dokumentiert den Aufbau einer JavaScript-basierten App-Registry mit mehreren Tool- und Lecture-Apps.

  • Zentrale App Registry (ohne JSON-Manifest)
  • Lecture-App mit CMS-ähnlicher Struktur
  • Tool-Apps (Währungsrechner, Buchstaben-Counter, Kontakte, Accordeon)
  • iframe-basierte Einbettung von Inhalten
